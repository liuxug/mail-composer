## 系统概述

**邮管家**是一款集成式邮箱管理系统，支持多邮箱账户统一管理，提供邮件收发、归档、搜索、标签等完整功能。系统采用前后端分离架构，前端使用 React + Tailwind CSS，后端使用 Node.js + Express，通过 WebSocket 实现新邮件实时推送。

### 主要功能

- 用户注册与登录
- 多邮箱账户管理（支持 QQ 邮箱、163 邮箱、126 邮箱、Gmail 等）
- 邮件收发与同步
- 邮件分类管理（收件箱、星标、已发送、草稿、归档、垃圾箱）
- 邮件搜索与筛选
- 富文本邮件编辑
- 标签管理
- 批量邮件操作
- 新邮件实时通知

## 系统安装与启动

### 后端启动

```bash
# 进入后端目录
cd backend

# 安装依赖
npm install

# 启动开发服务器（自动重启）
npm run dev

# 或启动生产服务器
npm start
```

后端服务默认运行在 `http://localhost:3000`

### 前端启动

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

前端开发服务器默认运行在 `http://localhost:5173`

## 用户认证

### 注册

1. 在登录页面点击 **"立即注册"** 切换到注册模式
2. 填写以下信息：
   - **用户名**：显示名称，必填
   - **邮箱地址**：用于登录和接收验证码，必填
   - **验证码**：点击"发送验证码"获取，必填
   - **密码**：至少6个字符，系统会显示密码强度提示
3. 点击 **"注册"** 完成注册

> 注意：验证码有效期为60秒，每个邮箱每分钟最多发送一次

### 登录

1. 在登录页面填写：
   - **邮箱地址**：注册时使用的邮箱
   - **密码**：注册时设置的密码
2. 点击 **"登录"** 进入系统

### 退出登录

1. 点击页面右上角的用户头像
2. 在弹出的菜单中选择 **"退出登录"**

## 邮箱账户管理

### 添加邮箱账户

1. 点击页面右上角的 **"添加邮箱"** 按钮，或点击左侧边栏的 **"添加邮箱"** 按钮
2. 在弹出的对话框中填写邮箱配置信息：
   - **邮箱地址**：完整的邮箱地址（如 `user@example.com`）
   - **SMTP 主机**：邮件发送服务器地址（如 `smtp.qq.com`）
   - **SMTP 端口**：邮件发送端口（通常为 465）
   - **IMAP 主机**：邮件接收服务器地址（如 `imap.qq.com`）
   - **IMAP 端口**：邮件接收端口（通常为 993）
   - **授权码**：邮箱的第三方应用授权码，**非登录密码**

3. 点击 **"添加"** 完成配置

> 提示：主流邮箱的配置信息如下：
> - QQ 邮箱：SMTP `smtp.qq.com` (465)，IMAP `imap.qq.com` (993)
> - 163 邮箱：SMTP `smtp.163.com` (465)，IMAP `imap.163.com` (993)
> - Gmail：SMTP `smtp.gmail.com` (465)，IMAP `imap.gmail.com` (993)

### 删除邮箱账户

1. 在左侧边栏找到要删除的邮箱账户
2. 点击账户右侧的 **删除图标**（垃圾桶图标）
3. 在确认对话框中点击 **"确定"**

> 注意：删除账户会同时删除该账户下的所有邮件数据，此操作不可恢复

### 切换邮箱账户

1. 在左侧边栏点击目标邮箱账户
2. 系统会自动切换到该账户的收件箱视图

## 邮件管理

### 邮件文件夹

系统提供以下默认文件夹：

| 文件夹 | 图标 | 说明 |
|--------|------|------|
| 收件箱 | 📥 | 接收的邮件 |
| 星标邮件 | ⭐ | 标记为重要的邮件 |
| 已发送 | 📤 | 已发送的邮件 |
| 草稿箱 | 📝 | 未发送的草稿 |
| 已归档 | 📦 | 已归档的邮件 |
| 垃圾箱 | 🗑️ | 已删除的邮件 |

### 查看邮件列表

1. 在左侧边栏选择邮箱账户和文件夹
2. 邮件列表会显示：
   - 发件人/收件人
   - 邮件主题
   - 邮件预览
   - 邮件时间
   - 星标状态
   - 附件标识

### 查看邮件详情

1. 点击邮件列表中的任意邮件
2. 在右侧详情面板中查看：
   - 邮件主题
   - 发件人/收件人信息
   - 发送时间
   - 邮件正文（支持 HTML 格式）
   - 附件列表
   - 标签

### 邮件搜索

1. 在页面顶部搜索框输入关键词
2. 按下 **Enter** 键执行搜索
3. 搜索结果会显示在邮件列表中

### 邮件筛选

1. 点击邮件列表上方的 **筛选按钮**（漏斗图标）
2. 设置筛选条件：
   - **状态**：全部/未读/已读
   - **是否有附件**：勾选后只显示带附件的邮件
   - **发件人**：按发件人邮箱筛选
   - **收件人**：按收件人邮箱筛选
   - **时间**：全部/今天/最近一周/最近一月/最近一年
   - **大小**：全部/小/中/大
3. 点击 **"确定"** 应用筛选

### 邮件操作

#### 星标邮件

- **方式一**：在邮件列表中点击邮件右侧的 **星标图标**
- **方式二**：在邮件详情页点击顶部的 **星标图标**

#### 删除邮件

- **移到垃圾箱**：点击邮件详情页的 **删除图标**，邮件会被移到垃圾箱
- **彻底删除**：在垃圾箱中选择邮件，点击 **"彻底删除"**，此操作不可恢复

#### 标记已读

- 在邮件列表中勾选邮件，点击 **"已读"** 按钮

#### 归档邮件

- 在邮件列表中勾选邮件，点击 **"归档"** 按钮
- 在已归档文件夹中可点击 **"移出归档"** 恢复到收件箱

### 批量操作

1. 在邮件列表中勾选多个邮件（点击复选框或全选按钮）
2. 选择要执行的操作：
   - **删除**：移到垃圾箱
   - **已读**：标记为已读
   - **归档**：移到归档文件夹
   - **标签**：添加标签
   - **移回收件箱**：从垃圾箱/归档恢复

### 邮件同步

1. 点击邮件列表上方的 **同步按钮**（刷新图标）
2. 系统会同步该账户下最近的邮件
3. 同步完成后会显示同步的邮件数量

## 写信功能

### 新建邮件

1. 点击页面顶部的 **"写信"** 按钮
2. 在写信界面填写：
   - **收件人**：邮件地址，多个用逗号分隔
   - **抄送**：可选，抄送人邮箱
   - **密送**：可选，密送人邮箱（收件人和抄送人不可见）
   - **主题**：邮件主题
   - **正文**：支持富文本编辑

### 富文本编辑

正文编辑器支持以下格式：

| 功能 | 图标 | 说明 |
|------|------|------|
| 撤销 | ↩️ | 撤销上一步操作 |
| 重做 | ↪️ | 重做上一步操作 |
| 加粗 | **B** | 文字加粗 |
| 斜体 | *I* | 文字斜体 |
| 下划线 | U̲ | 文字下划线 |
| 删除线 | ~~S~~ | 文字删除线 |
| 无序列表 | ☰ | 项目符号列表 |
| 有序列表 | ≡ | 数字编号列表 |
| 左对齐 | ⬅️ | 段落左对齐 |
| 居中对齐 | ⬜ | 段落居中 |
| 右对齐 | ➡️ | 段落右对齐 |
| 插入链接 | 🔗 | 插入超链接 |
| 插入图片 | 🖼️ | 插入图片（URL） |
| 插入表格 | 📊 | 插入表格 |
| 分割线 | ➖ | 插入水平分割线 |
| 引用 | ❝ | 引用格式 |

### 添加附件

1. 点击 **"添加附件"** 按钮
2. 选择要上传的文件（支持多个文件）
3. 附件会显示在附件列表中，可点击 **删除图标** 移除

### 回复邮件

1. 在邮件详情页点击 **"回复"** 按钮
2. 系统会自动填写：
   - 收件人：原始发件人
   - 主题：自动添加 "Re:" 前缀
   - 正文：包含原始邮件引用

### 回复全部

1. 在邮件详情页点击 **"回复全部"** 按钮
2. 系统会自动填写所有收件人（包括原始发件人、收件人、抄送）

### 转发邮件

1. 在邮件详情页点击 **"转发"** 按钮
2. 系统会自动填写：
   - 主题：自动添加 "Fwd:" 前缀
   - 正文：包含原始邮件引用

### 保存草稿

1. 在写信界面点击 **"存草稿"** 按钮
2. 邮件会保存到草稿箱，可稍后继续编辑

### 发送邮件

1. 填写完邮件内容后点击 **"发送"** 按钮
2. 邮件会通过配置的 SMTP 服务器发送

### 切换发件人

1. 在写信界面底部点击 **"更换发件人"**
2. 在弹出的列表中选择其他邮箱账户作为发件人

## 常见问题

### Q: 邮箱添加失败怎么办？

A: 请检查以下几点：
1. 确认邮箱地址正确
2. 确认 SMTP/IMAP 服务器地址和端口正确
3. 确认授权码正确（**非登录密码**）
4. 确认邮箱已开启 IMAP/SMTP 服务

### Q: 如何获取邮箱授权码？

A: 不同邮箱的获取方式略有不同：
- **QQ 邮箱**：设置 → 账户 → 开启 IMAP/SMTP 服务 → 获取授权码
- **163 邮箱**：设置 → POP3/SMTP/IMAP → 开启 IMAP/SMTP 服务 → 获取授权码
- **Gmail**：设置 → 转发和 POP/IMAP → 开启 IMAP → 在账户安全中生成应用密码

### Q: 邮件同步很慢或同步失败？

A: 可能原因：
1. 网络连接不稳定
2. 邮箱服务器限制（如 QQ 邮箱限制单次同步数量）
3. 邮件数量过多

### Q: 新邮件没有实时推送？

A: 请检查：
1. 浏览器是否允许 WebSocket 连接
2. 网络连接是否稳定
3. 可尝试刷新页面重新建立连接

### Q: 附件无法下载？

A: 请检查：
1. 网络连接是否正常
2. 文件大小是否超过系统限制
3. 文件是否被浏览器拦截
